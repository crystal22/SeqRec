## Overview

There are some source codes of SOTA models for sequential recommendation, which are implemented by TensorFlow.

## Model Introduction

- **PopRec**: recommend items according to their popularity.
- **BPRMF[2009]**: [<< BPR: Bayesian Personalized Ranking from Implicit Feedback >>](https://arxiv.org/ftp/arxiv/papers/1205/1205.2618.pdf)
- **FPMC[2010]**: << Factorizing Personalized Markov Chains for Next-Basket Recommendation >>
- **GRU4Rec[2016]**: [<< SESSION-BASED RECOMMENDATIONS WITH RECURRENT NEURAL NETWORKS>>](https://arxiv.org/pdf/1511.06939.pdf)
- **Caser[2018]**: [<< Personalized Top-N Sequential Recommendation via Convolutional Sequence Embedding >>](https://arxiv.org/pdf/1809.07426.pdf)
- **SASRec[2018]**: [<< Self-Attentive Sequential Recommendation >>](https://arxiv.org/pdf/1808.09781.pdf)
- **NextItNet[2019]**: [<< A Simple Convolutional Generative Network for Next Item Recommendation >>](https://arxiv.org/pdf/1808.05163.pdf)

- **HGN[2019]**: [<< Hierarchical Gating Networks for Sequential Recommendation >>](https://arxiv.org/pdf/1906.09217.pdf)
